# 🐍 Python Mini Projects Collection

Welcome to my **Python Projects Repository**! 🚀
This repository contains a collection of beginner-friendly Python projects that demonstrate core programming concepts, logic building, and problem-solving skills.

## 📌 Projects Included

### 🧮 1. Rent Calculation System

* Calculates total rent based on inputs like:

  * Room rent
  * Food expenses
  * Electricity usage
  * Number of people
* Helps in fair expense distribution.

---

### ✊✋✌️ 2. Rock Paper Scissors Game

* A simple CLI-based game.
* Play against the computer.
* Uses randomization and conditional logic.

---

### ❌⭕ 3. Tic Tac Toe Game

* Classic 2-player game.
* Built using Python (can be CLI or Tkinter GUI).
* Demonstrates:

  * Game logic
  * Win conditions
  * User interaction

---

### 🕒 4. Digital Clock

* Displays real-time clock.
* Built using Python and Tkinter.
* Uses system time functions.

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/your-repo-name.git
   ```

2. Navigate to the project folder:

   ```bash
   cd your-repo-name
   ```

3. Run any project:

   ```bash
   python filename.py
   ```
